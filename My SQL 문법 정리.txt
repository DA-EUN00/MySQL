<My SQL 특징>
서버와의 연결을 끊는 구문인 QUIT와 같은 경우를 제외한 일반적인 구문 뒤에는 세미콜론(;)을 붙임. (이러한 세미콜론은 SQL 구문을 구분하는 기준)

MySQL은 키워드와 구문에서 대소문자를 구분하지 않음.
But, 테이블 명과 필드의 이름은 대소문자를 구분!!
*[]는 보기 좋으라고 필자가 임의로 넣은 것


<My SQL 주석>
1. # 한 줄 주석
2. -- 한 줄 주석
3. /* 두 줄
   이상의
   주석 */

CREATE DATABASE [데이터베이스이름];

<데이터베이스의 선택> 
USE [데이터베이스이름];

<테이블 생성>
CREATE TABLE [테이블이름]
(
    [필드이름1] [필드타입1],
    [필드이름2] [필드타입2],
    ...
);
